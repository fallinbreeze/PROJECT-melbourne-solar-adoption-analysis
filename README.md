# Melbourne Solar Adoption Analysis

## Project Overview

This project investigates residential solar adoption patterns across Melbourne postcodes using housing, solar installation, and ABS Census data.

## Business Objective

To help ARENA identify postcode-level characteristics associated with solar adoption and support future solar program planning.

## Methods

- Data Cleaning and Transformation
- Aggregation to Postcode Level
- K-Means Clustering
- OLS Regression Analysis
- Interactive Mapping

## Data Sources

- ARENA Solar Installation Data
- Melbourne Housing Market Data
- ABS Census 2021

## Key Findings

- Property characteristics influence solar adoption.
- Distance from CBD shows a significant relationship with solar output.
- Different postcode groups exhibit distinct solar adoption patterns.
